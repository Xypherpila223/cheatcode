# cheatcode

## Data Analytics "CheatCodes" & Workflow Optimization

This repository serves as a centralized documentation hub for the high-efficiency shortcuts, SQL snippets, and Excel macros I use to streamline the data lifecycle. From initial data cleaning to final visualization, these "cheat codes" are designed to reduce manual input lag and ensure technical accuracy across complex datasets.

Key Focus Areas:

SQL Snippets: Optimized queries for rapid data extraction and validation.

Excel/Google Sheets Macros: Keyboard shortcuts and formulas for advanced data manipulation.

Visualization Hotkeys: Techniques for faster dashboarding in Tableau and Looker Studio.

Process Optimization: Documentation on improving data entry and processing workflows
